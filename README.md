# ThemeHunter 题材猎手

**A股新题材发现与埋伏系统**

---

## 🎯 项目定位

ThemeHunter 是一个独立的A股题材分析系统，专注**新题材发现**和**提前埋伏**策略。

### 核心原则

| ✅ 做 | ❌ 不做 |
|-------|--------|
| 新题材发现（<30天） | 旧题材（已炒作>30天） |
| 阶段判断 | 交易执行 |
| 演化预测 | 模拟账户 |
| 标的推荐 | 持仓管理 |
| 埋伏建议 | 止损止盈 |
| 报告推送 | 资金管理 |

## 🔑 核心功能

### 1. 新题材发现

**只关注新题材**（30天内），自动过滤旧题材：

| 指标 | 新题材 | 旧题材 |
|------|--------|--------|
| 首次提及 | < 30天 | > 30天 |
| 龙头涨幅 | < 50% | > 50% |
| 热度趋势 | ⬆️ 上升 | ⬇️ 下降 |
| 资金流向 | 流入 | 流出 |

**题材生命周期**：

```
萌芽期 ──→ 爆发期 ──→ 炒作期 ──→ 退潮期
  ↓           ↓           ↓           ↓
 埋伏点     追涨点      高风险      离场点
 推荐买入   可追涨      谨慎参与    坚决回避
```

### 2. 题材演化链追踪

系统能识别题材演化路径，预判下一阶段热点：

**疫情类演化链**：
```
防护物资 ──→ 检测试剂 ──→ 疫苗研发 ──→ 治疗药物 ──→ 复工复产
   ↓            ↓            ↓            ↓
 口罩/防护服   核酸/抗原    科兴/康希诺   阿兹夫定
```

**新能源演化链**：
```
政策驱动 ──→ 技术突破 ──→ 产业化 ──→ 业绩兑现
   ↓            ↓            ↓
 补贴政策    量产/成本    订单落地    财报增长
```

**AI演化链**：
```
技术突破 ──→ 应用落地 ──→ 生态建设 ──→ 业绩兑现
   ↓            ↓            ↓
 GPT-4发布   B端/C端应用   开发者生态   收入增长
```

**消费电子演化链**：
```
产品发布 ──→ 量产爬坡 ──→ 销售旺季
   ↓            ↓            ↓
 发布会       产能/良率    销量/份额
```

### 3. 阶段判断

系统对每个题材判断当前所处阶段：

| 阶段 | 特征 | 建议 |
|------|------|------|
| 🌱 萌芽期 | 刚出现苗头，市场认知度低 | ⭐⭐⭐⭐⭐ 最佳埋伏时机 |
| 🚀 爆发期 | 热度快速上升，龙头启动 | ⭐⭐⭐⭐ 可追涨龙头 |
| 🔥 炒作期 | 全面扩散，资金涌入 | ⭐⭐ 高风险，谨慎参与 |
| 📉 退潮期 | 热度下降，资金流出 | ❌ 坚决回避 |

### 4. 标的挖掘

每个题材推荐两类标的：

**龙头股**（核心标的）：
- 行业地位领先
- 技术壁垒高
- 资金认可度高
- 题材关联度强

**跟风股**（弹性标的）：
- 弹性较大
- 股性活跃
- 适合波段操作

### 5. 催化剂追踪

识别题材的引爆点：
- 政策发布（如：低空经济示范区名单）
- 产品发布（如：苹果Vision Pro）
- 会议事件（如：华为开发者大会）
- 技术突破（如：固态电池量产）
- 数据发布（如：销量超预期）

## 🤖 Agent系统

### 6大专用Agent

| Agent | 权重 | 职责 | 核心能力 |
|-------|------|------|----------|
| 📜 政策解读师 | 1.3 | 解读政策文件 | 提取利好方向、时间节点 |
| 📰 新闻猎手 | 1.2 | 抓取热点新闻 | 识别题材萌芽信号 |
| 🔬 技术前瞻者 | 1.1 | 跟踪技术突破 | 预判产业化时机 |
| 📅 事件策划师 | 1.0 | 分析事件时间线 | 会议、展会、发布会 |
| 🎯 标的挖掘机 | 1.2 | 挖掘相关股票 | 龙头/跟风/边缘区分 |
| ⏰ 题材周期师 | 1.4 | 判断题材阶段 | 萌芽/爆发/炒作/退潮 |

### Agent协作流程

```
新闻采集 → 政策解读师/新闻猎手/技术前瞻者
    ↓
题材提取 → 题材周期师判断阶段
    ↓
演化预测 → 事件策划师识别催化剂
    ↓
标的挖掘 → 标的挖掘机推荐股票
    ↓
综合评分 → 加权投票得出最终建议
```

```yaml

  version: "1.0.0"
```
